HTML

Hypertext Markup Language (HTML) 이란 웹페이지와 웹애플리케이션을 만드는 데 사용되는 기초 마크업(문서정형) 언어입니다.

웹브라우저는 웹 서버나 기기의 저장소에서 HTML을 받아 멀티미디어 웹 페이지로 구현합니다.
HTML은 웹페이지의 구조를 의미있게 묘사하며 문서의 생김새를 표현하는 정보를 내재하고 있습니다.

HTML 요소는 HTML 페이지를 쌓아올리는 블럭과 같은 존재입니다.
HTML 구조가 이미지나 다른 물체와 같이 사용자와 상호작용하는 형태들을 페이지에 내장하도록 하는 것입니다.
이것은 구조적 의미를 해독하여 텍스트를 머릿글이나 문단, 목록, 링크, 인용 등 다양한 항목을 추가할 수 있게 함으로써
잘 짜여진 문서를 창조해내는 데 도움이 됩니다.

HTML 요소는 꺽쇠를 사용한 태그로 기술됩니다.
